Welcome to Soil Companion

This is a chatbot for the SoilWise project.

Ask soil-related questions and get answers based on the open access knowledge and data repository that the project collects.

SoilWise aims to foster soil wisdom, advance agriculture, and build a legacy of greener, healthier soils.

The chat memory and any uploaded file will be erased 15 minutes after your last message. No information is kept or used for training models.

Getting started

  1. Ask a question related to soil, agriculture, or soil health. Example: How do cover crops improve soil structure?
  2. Wait for the bot to respond. You can ask follow-up questions to refine the answer.
  3. Rate answers with the thumbs-up/down icons to help improve future responses.

Tips

  • You can ask questions in languages other than English.
  • Ask one thing at a time; follow up for details.
  • Questions are limited to 400 characters, so keep them focused.
  • Context from earlier in the conversation is remembered within a session.
  • The Clear button starts a fresh session, resetting conversation memory.
  • Set a location in the sidebar for region-specific answers.
  • Ask the bot to search the catalog for datasets or publications if needed.
  • Check the Insight tab for related topics and Wikipedia links.
  • Ask for a table, bullet list, summary, or writing style to shape the response.
