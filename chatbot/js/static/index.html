This is a chatbot for the SoilWise project.


Ask it a soil related question, and you'll get answers based on the open access knowledge and data repository that the project collects.


SoilWise aims to foster soil wisdom, advance agriculture, and build a legacy of greener, healthier soils. The chatbot will prefer giving answers that take these aims into account.


The chat memory and any uploaded file will be erased 15 minutes after your last message. No information is kept or used for training models.


You can voluntarily provide feedback by rating answers given by the chatbot, or by contacting SoilWise via this form.

Getting started

  1. Ask a question related to soil, agriculture, or soil health. Example: How do cover crops improve soil structure?
  2. Wait for the AI to respond. You can ask follow-up questions to refine the answer.
  3. Rate answers with the thumbs-up/down icons to help improve future responses.

Tips

  • Select a location in the sidebar for more specific answers.
  • Ask one thing at a time for clearer results.
  • Instruct the AI to look in the SoilWise catalog for datasets and papers if it doesn't by itself.
  • Or ask it to provide references to catalog items if they are not provided.
  • You can upload a temporary text file with specific information, e.g., a soil sampling report.
  • Use the Copy button at the top right to copy the conversation.
  • The Clear button can be used to start afresh.

Disclaimer


The chatbot’s responses should not be considered as official agricultural advice, validated soil data, or policy guidance. For professional or field-level decisions, please consult authoritative data sources or qualified experts.
